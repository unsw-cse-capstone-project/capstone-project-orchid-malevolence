# ReadMe

## 1、 Install Node.js

```bash
# windows install
https://nodejs.org/en/download/

# ubuntu & macOS install
sudo apt-get install nodejs
sudo apt-get install npm
```

## 2、 Install vue-cli

```bash
npm install vue-cli -g
```

## 3、 Go to the project form

```bash
cd mainpage
```
## 4、 Run serve

```bash
npm run serve
```
