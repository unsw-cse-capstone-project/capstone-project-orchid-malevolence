# ReadMe

## 1、 Install Node.js

```
https://nodejs.org/en/download/
```

## 2、 Install vue-cli

```bash
npm install vue-cli -g
```

## 3、 Go to the project form

```bash
cd my-project
```
## 4、 Install jquery bootstrap popper.js

```bash
npm install jquery bootstrap popper.js --save -dev
```

## 5、 Run serve

```bash
npm run serve
```
